PLAYER COUNT 6 FIX
==================

This project is already set to 6 players.

The Change Player Count tool was repaired so it always edits the exact project
folder that contains the setup_powershell folder. It no longer chooses an older
remembered copy from Downloads.

To publish the 6-player version:
1. Run setup_powershell\Deploy Firestore Rules.cmd
2. Run setup_powershell\Update Entire Project to GitHub.cmd
3. Wait about one minute.
4. Press Ctrl+F5 on the website.
5. Cancel any old match and create a new match.

Existing matches keep the player count they were created with.
